FHIR FLI version 2.0

Real-time, FHiR based & decentralized health data analytics dashboard

Get Started

Abstract

FHIR FLI Version 2.0 is a FHiR based platform for managing fitness and lifestyle data. It manages this data in a well-known standard called FHiR, which is a draft standard describing data formats and elements for exchanging electronic health records. The user is able to input his data from the Fitbit smart watch, then through a decentralised storage system, his data is stored. The user will then be able to view visualisations and statistical graphs of his data on his dashboard. Moreover, the user can associate his account with a company if needed, add health goals such as losing weight that are going to be tracked through his data. Finally, there is also a login feature for corporatations.
