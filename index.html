Introduction

This project is about solving the problem of scattered health and lifestyle data, the solution is creating a personalized data dashboard for the users to have there data in one place where they can access it easily and see different analysis graphs.

Key features

  1. User registeration system



  2. User Dashboard for viewing various health data as blood pressure, BMI & heart rate



  3. Fitbit data being shown is a visualised way, as graphs
