Get Started

Introduction

FHiR FLI 2.0 is an open-source health data visualization platform. On the user end, users can sync their data from their Fitbit watches directly into the website in real-time, this will allow them to make efficient use of their health & life style data over long period of time. On the technical end, the data has been converted to a FHiR based data schema - a health-data framework which has been gaining popularity in many health organizations for transferring medical records.

Key features


  1. User registeration system



  2. User Dashboard for viewing various health data as blood pressure, BMI & heart rate



  3. Fitbit data being shown is a visualised way, as graphs
