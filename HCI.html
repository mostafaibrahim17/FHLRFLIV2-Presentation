Use cases


Use Case Name Find User Registeration Form
Use Case 1
Description A user open the FHIR FLI web application to sign up.
Primary Actors User
Secondary Actors System
Pre-conditions None
Main Flow 1. User presses on the "Go To Individual Sign In" button at the bottom-right corner of the screen or presses on the "Go To Corporate Sign In" button at the bottom-left corner of the screen
2. User click the "Sign Up" button at the screen center
3. User enters his email and password
4. User presses register and his done.
Post-Conditions None
Alternative Flow None


Use Case Name Find User Login Form
Use Case 2
Description A user open the FHIR FLI web application to login using his/her credentials.
Primary Actors User
Secondary Actors System
Pre-conditions User must have an account on FHIR FLI 2.0
Main Flow 1. User presses on the "Go To Individual Sign In" button at the bottom-right corner of the screen or presses on the "Go To Corporate Sign In" button at the bottom-left corner of the screen
2. User click enter his/her credentials
3. User clicks the login button
Post-Conditions None
Alternative Flow None


Use Case Name Find visualisations on the user's dashboard
Use Case 3
Description A user open the FHIR FLI web application to view his/her visualisations
Primary Actors User
Secondary Actors System
Pre-conditions User must have an account on FHIR FLI 2.0
Main Flow 1. User presses on the "Go To Individual Sign In" button at the bottom-right corner of the screen or presses on the "Go To Corporate Sign In" button at the bottom-left corner of the screen
2. User enter his/her credentials and logs in
3. If login is successful user will see the visualisations
Post-Conditions None
Alternative Flow None
